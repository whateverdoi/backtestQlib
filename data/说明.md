# 数据的下载和聚合使用说明
- 在[**这里**](https://data.binance.vision/)下载数据,注意数据是aggtrades或者trades,而不是kline
- 使用rust写好的工具bars聚合数据具体代码和使用请看[**仓库地址**](https://github.com/whateverdoi/bars)
    1.时间k线,
    ```
    ./bars --input aggtrades.csv --output timebars.csv --bar-type time1-m
    ```
    2.dollar Bar
    ```
    ./bars --input aggtrades.csv --output dollarbars.csv --bar-type dollar
    ```
    3.Volume Run Bar：
    ```
    ./bars --input aggtrades.csv --output volumerunbars.csv --bar-type volume-run
    ```
